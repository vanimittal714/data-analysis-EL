# Data Cleaning Task 1

## 📊 Dataset
Netflix Movies and TV Shows Dataset (from Kaggle)

## 🛠 Tools Used
- Microsoft Excel

## 🧹 Data Cleaning Steps
- Checked and removed duplicate records (none found)
- Identified missing values using filters
- Filled missing values in:
  - Director → "Not Available"
  - Cast → "Not Available"
  - Country → "Unknown"
- Converted date_added column to proper date format
- Standardized column names (lowercase, underscores)
- Checked for inconsistent text values
- Ensured correct data types

## ✅ Final Result
A clean and structured dataset ready for analysis.
